TouchZone.cs
	Для быстрого редактирования зоны стиков:
	1. Добавляем на любой объект скрипт TouchZone
	2. Заполняем поля самих зон (Left Joystick Zone и Right Joystick Zone - находим в иерархии)
	3. Включаем галку "Edit In Scene" и дергаем ползунки
	4. После изменений галку желательно выключить

MoveJoystick.cs и ShootJoystick.cs
	Галка "Smooth Enabled" подразумевает надо ли использовать плавное появление и затухание стика
	Если включена - поля "Smooth Curve" и "Transition Duration" надо заполнить
	"Smooth Curve" - кривая используется для нелинейного сглаживания
	"Transition Duration" - время за которое появляется/затухает стик
	Для горизонтального передвижения можно использовать клавиши A/D

Boom.cs
	Используется для мини-игры после запуска
	Чем больше значение fillAmount в Image - тем большей будет отталкивающая сила (на данный момент только сообщение в консоль)
	Можно поиграться с кривыми для более неожиданной реакции
